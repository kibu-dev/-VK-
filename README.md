# 🤖 Бот для паблика «Подслушано»

Автоматическая публикация постов из предложок + бот для личных сообщений (статистика, удаление постов, поддержка, модерация).

## 📋   Возможности

- ✅ Автоматическая публикация обычных постов (без ссылок)
- ✅ Подозрительные посты (ссылки, запрещённые слова) → уведомление админу
- ✅ Админ может опубликовать или удалить подозрительный пост командами
- ✅ Анонимность по ключевым словам (анон, анонимно, #анон)
- ✅ Подпись автора в конце поста (кликабельная ссылка или «Аноним»)
- ✅ Удаление своих постов пользователями через ЛС
- ✅ Поддержка пользователей (пересылка сообщений админу)
- ✅ Чёрный список запрещённых слов (админ добавляет/удаляет)
- ✅ Интервал между публикациями (настраивается)

                     # Инструкция


🔑 Как получить токены

USER_TOKEN (токен пользователя)

1. Перейди на vkhost.github.io
2. Выбери права:
   · ✅ Управление сообществом
   · ✅ Доступ к стене
   · ✅ Доступ к сообщениям
3. Нажми «Получить ключ»
4. Скопируй полученную строку (начинается с vk1.a...)

GROUP_TOKEN (токен сообщества)

1. Зайди в Управление сообществом → Работа с API
2. Нажми «Создать ключ»
3. Выбери права:
   · ✅ Сообщения сообщества
   · ✅ Управление сообществом
4. Скопируй токен

ID группы и свой ID

· ID группы: из адреса vk.com/club123456789 → 123456789
· ADMIN_ID: перейди на id.vk.com

🔧 Команды для администратора (только в ЛС)

Команда Действие Пример
!pub 123 Опубликовать подозрительный пост !pub 179
!del 123 Удалить подозрительный пост !del 179
!addw слово Добавить слово в чёрный список !addw спам
!delw слово Удалить слово из чёрного списка !delw спам
!listw Показать все запрещённые слова !listw

---

⚙️ Настройка интервалов

В файле .env:

Переменная По умолчанию Что делает
CHECK_INTERVAL 60 Как часто проверять предложки (сек)
PUBLISH_INTERVAL 600 Интервал между публикациями (сек)

🎯 Анонимность

Пользователь может добавить в пост ключевые слова:

· анон
· анонимно
· аноним
· #анон
· #анонимно
· #аноним

Пост опубликуется как есть + в конце добавится подпись «Автор: Аноним».

---

🛡️ Чёрный список слов

Посты с запрещёнными словами не публикуются, админ получает уведомление.

Админ может управлять списком командами:

· !addw слово — добавить слово
· !delw слово — удалить слово
· !listw — посмотреть все слова
